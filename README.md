# Blooming

> Growth that matters. / Crecimiento que importa. / Une croissance qui compte.

## Stack

- **Next.js 14** — App Router
- **TypeScript**
- **Tailwind CSS**
- **Fonts:** Cormorant Garamond (display) + Space Grotesk (body)

## Structure

```
/app
  /page.tsx           ← public landing page
  /layout.tsx         ← global layout + fonts
  /(dashboard)
    /page.tsx         ← internal workspace (placeholder)
/public
  /logo.svg           ← brand mark placeholder
/components
  /ui/                ← base UI components
```
